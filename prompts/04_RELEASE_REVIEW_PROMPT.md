# Prompt — Release review URIM / Kairos

Active `prediction-release-gate` et `security-review`.

Réalise un audit PASS/FAIL :
- intégrité temporelle ;
- provenance ;
- données réelles vs fixtures ;
- calibration ;
- tests ;
- ledger immuable ;
- sécurité ;
- observabilité ;
- conformité ;
- langage produit.

Un seul bloquant implique FAIL. Donne preuves, chemins, commandes et corrections minimales.
