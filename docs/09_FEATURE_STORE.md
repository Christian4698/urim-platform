# Feature Store

## Exigences
Features versionnées, déterministes, avec `as_of`, dépendances, TTL, fraîcheur, tests de non-fuite, missing explicite et suivi du drift.

## Exemples
Forme ajustée à l’opposition, buts/xG domicile-extérieur, repos, disponibilité, stabilité tactique, tirs, déplacement, météo, mouvement de cote horodaté.

## Anti-patterns
Classement final, moyenne incluant le match cible, cote après coup d’envoi, label encodé, imputation apprise sur l’ensemble complet.
