# Glossaire

- **Observation** : valeur reçue d’un fournisseur avec horodatage et provenance.
- **Disponible à** : première heure à laquelle l’information pouvait être utilisée.
- **Snapshot** : ensemble immuable des données utilisées pour une prédiction.
- **Feature** : variable calculée uniquement avec les informations disponibles à l’instant de décision.
- **Calibration** : concordance entre probabilités annoncées et fréquences observées.
- **Couverture** : proportion de matchs recevant un conseil.
- **No-Bet** : refus explicite.
- **Look-ahead bias** : utilisation d’informations futures.
- **Target leakage** : variable révélant directement ou indirectement la cible.
- **Provider reconciliation** : résolution des contradictions entre fournisseurs.
- **Canonical ID** : identifiant interne stable.
