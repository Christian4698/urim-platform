# Outils et extensions

## Phase documentation/architecture
**Aucune extension n’est obligatoire.**

## Outils locaux recommandés
Git, Docker Compose, Python 3.12+, `uv` ou Poetry, Node.js LTS, `pnpm`, client PostgreSQL, Make/Taskfile.

## Extensions VS Code recommandées
Python, Pylance, Ruff, Docker, PostgreSQL, ESLint, Prettier, GitLens, REST Client ou Thunder Client, YAML et EditorConfig.

## MCP optionnels
GitHub, Supabase/PostgreSQL et observabilité selon la stack.

Codex doit préciser dans chaque plan les outils réellement nécessaires et signaler explicitement lorsqu’aucune extension supplémentaire n’est requise.
