# Réponse aux incidents

## Niveaux
- SEV-1 : fuite temporelle en production, secret exposé, données falsifiées ou prédictions réécrites.
- SEV-2 : fournisseur critique indisponible, live très retardé ou drift majeur.
- SEV-3 : couverture partielle ou écran dégradé.

## Processus
Suspendre, préserver les preuves, identifier la période touchée, révoquer les secrets si besoin, marquer sans supprimer, corriger avec tests, publier un post-mortem et mettre à jour les instructions.
